# SmartClipboard Pro v7.2

**SmartClipboard Pro**는 Windows 환경을 위한 강력하고 현대적인 클립보드 매니저입니다. 복사한 모든 텍스트, 이미지, 링크를 자동으로 저장하고 안전하게 관리하세요.

## ✨ 주요 기능 (v7.2 Update)

### 🚀 핵심 기능
*   **히스토리 자동 저장**: Ctrl+C로 복사한 모든 내용을 로컬 데이터베이스에 저장합니다.
*   **스마트 분류**: 텍스트, 이미지, 링크, 코드, 색상 등을 자동으로 분류합니다.
*   **강력한 검색**: 키워드 검색 및 태그 필터링으로 과거 기록을 즉시 찾을 수 있습니다.
*   **고정(Pin) & 즐겨찾기**: 자주 쓰는 항목을 상단에 고정하거나 순서를 변경할 수 있습니다.

### 🛡️ 보안 및 안정성 (New!)
*   **🔒 프라이버시 모드**: 비밀번호 등 민감한 작업 시 클립보드 저장을 일시적으로 중단할 수 있습니다.
*   **💾 데이터 백업/복원**: 소중한 클립보드 기록을 .db 파일로 백업하고 복원하는 기능을 제공합니다.
*   **안전한 데이터 관리**: 전원 차단 시에도 데이터가 손상되지 않도록 트랜잭션 처리가 강화되었습니다.

### 🎨 UI/UX 개선
*   **현대적인 디자인**: 다크 모드, 라이트 모드, 오션 모드 등 다양한 테마를 지원합니다.
*   **Toast 알림**: 복사 시 깔끔한 슬라이드 애니메이션 알림을 제공합니다.
*   **편의 기능**:
    *   **텍스트 변환**: 대소문자 변환, 공백 제거, JSON 포맷팅 등.
    *   **툴팁 미리보기**: 긴 텍스트도 마우스 오버로 전체 내용을 빠르게 확인.
    *   **QR 코드 생성**: 링크나 텍스트를 즉시 QR 코드로 변환.

## ⌨️ 키보드 단축키

| 단축키 | 동작 |
| :--- | :--- |
| **Ctrl + Shift + V** | 프로그램 창 열기/숨기기 (글로벌) |
| **Ctrl + C** | 선택 항목 복사 |
| **Enter** | 선택 항목 붙여넣기 (창 닫힘) |
| **Delete** | 선택 항목 삭제 |
| **Shift + Delete** | 선택 항목 영구 삭제 |
| **Ctrl + P** | 항목 고정/해제 토글 |
| **Ctrl + F** | 검색창으로 포커스 이동 |
| **Esc** | 검색어 지우기 / 창 닫기 |
| **Ctrl + Q** | 프로그램 종료 |

## 📦 설치 및 실행 방법

### 요구 사항
*   Windows 10/11
*   Python 3.9 이상 (소스 코드 실행 시)

### 1. 실행 파일 빌드 (권장)
PyInstaller를 사용하여 단독 실행 파일(.exe)을 생성할 수 있습니다.

```powershell
pip install -r requirements.txt
pyinstaller smartclipboard.spec
```
빌드가 완료되면 `dist/SmartClipboardPro.exe` 파일이 생성됩니다.

### 2. 소스 코드 실행
```powershell
# 가상 환경 생성 및 활성화
python -m venv venv
.\venv\Scripts\activate

# 의존성 설치
pip install PyQt6 keyboard pywin32 qrcode pillow

# 실행
python "클립모드 매니저.py"
```
