# SmartClipboard Pro v6.3

**SmartClipboard Pro**는 Windows용 고급 클립보드 매니저입니다. 복사한 텍스트, 이미지, 링크 등을 자동으로 저장하고 관리하며, 강력한 태그 시스템과 검색, 고정 기능을 제공합니다.

## ✨ 주요 기능

*   **📋 히스토리 관리**: 클립보드에 복사된 모든 내용을 자동으로 저장하고 관리합니다.
*   **📌 항목 고정**: 자주 사용하는 항목을 상단에 고정하여 빠르게 접근할 수 있습니다.
*   **🖼️ 이미지 지원**: 텍스트뿐만 아니라 이미지 클립보드도 저장하고 미리볼 수 있습니다.
*   **🏷️ 태그 시스템**: 항목에 태그를 지정하여 효율적으로 분류하고 필터링할 수 있습니다.
*   **📝 스니펫 관리**: 자주 쓰는 상용구를 등록하고 템플릿 변수(날짜, 시간 등)를 사용하여 빠르게 붙여넣을 수 있습니다.
*   **⚙️ 복사 규칙**: 정규식 패턴을 기반으로 복사한 텍스트를 자동으로 변환(공백 제거, 대소문자 변환 등)하거나 특정 동작을 수행합니다.
*   **🎨 테마 지원**: 다크, 라이트, 오션 등 다양한 테마를 지원하여 취향에 맞게 사용할 수 있습니다.
*   **🔒 안전한 저장**: 모든 데이터는 로컬 SQLite 데이터베이스에 안전하게 저장됩니다.
*   **⌨️ 단축키 지원**: 글로벌 단축키로 언제 어디서나 빠르게 창을 호출하고 사용할 수 있습니다.

## 🚀 설치 및 실행

### 요구 사항
*   Windows 10/11
*   (소스 코드 실행 시) Python 3.9+

### 실행 방법
1.  **배포판**: `SmartClipboardPro.exe`를 실행합니다.
2.  **트레이 아이콘**: 프로그램은 실행 시 시스템 트레이에 상주합니다. `Ctrl+Shift+V`를 누르면 창이 나타납니다.

## ⌨️ 단축키

| 키 | 동작 |
| :--- | :--- |
| **Ctrl+Shift+V** | 창 표시/숨기기 (글로벌) |
| **Ctrl+C** | 선택 항목 복사 (테이블 포커스 시) |
| **Enter** | 선택 항목 붙여넣기 (창 숨김 + Ctrl+V) |
| **Delete** | 선택 항목 삭제 |
| **Ctrl+P** | 선택 항목 고정/해제 |
| **Ctrl+F** | 검색창으로 포커스 이동 |
| **Esc** | 검색어 지우기 또는 창 숨기기 |

## 🛠️ 개발 및 빌드

### 의존성 설치
```bash
pip install -r requirements.txt
# 또는
pip install PyQt6 keyboard pillow qrcode
```

### 빌드 (PyInstaller)
```bash
pyinstaller smartclipboard.spec
```
